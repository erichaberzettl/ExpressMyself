## Short description

Learn practical expressions from a lightweight popup and searchable phrase library.

## Detailed description

ExpressMyself helps you build a more natural feel for everyday language through short, useful phrase cards.

Use the popup to get a daily expression, hear pronunciation with speech playback, and save phrases you want to revisit later. Open the full extension page to browse a larger library, filter by language or topic, and build a compact saved list for review.

Current features:

- daily phrase rotation
- support for 10 languages
- searchable phrase library
- topic filters
- saved phrases stored in the browser
- speech playback for expressions
