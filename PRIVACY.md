# ExpressMyself Privacy Policy

Last updated: April 25, 2026

ExpressMyself is a lightweight language-learning app and Chrome extension for browsing idioms, expressions, and everyday phrases.

## What data the extension stores

The Chrome extension stores only:

- your selected language
- the list of expressions you save for later review

This information is stored locally in your browser using Chrome extension storage.

## What data the extension does not collect

ExpressMyself does not collect:

- account information
- names
- email addresses
- payment information
- browsing history
- page contents
- keystrokes
- location data

## Data sharing

ExpressMyself does not sell, transfer, or share personal data with third parties.

## Remote services

The extension does not require a backend service to function and does not send saved-expression data to a remote server.

If you publish this extension publicly, replace this section with your support email or website before submitting to the Chrome Web Store.
